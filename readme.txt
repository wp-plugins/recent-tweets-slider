### Recent Tweets Slider

== Installation ==

### Installation

1. Upload 'recent-tweets-slider' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Appearance > Widgets > Recent Tweets Slider.
4. [How to get your Twitter Consumer Key & Consumer Secret](http://wp-time.com/recent-tweets-slider-plugin/)
